## 一.搜索

### 1. 深度优先搜索（非图论dfs）

本质上是一种枚举（优雅的枚举），时间复杂度较高，可以通过剪枝的方法进行优化。

基本结构：

``` C++
void dfs(int step)
{
    //判断是否满足题目条件
    if (/*结果判断*/)
        return;
    //循环枚举待验证解
    for(int i ; i < num ; i++)
    {
        mark[i] = true;//标记枚举过的节点
        dfs(step+1);//进行下一层枚举
        mark[i] = false;//还原枚举过的节点
    }
    return;
}
```
